# User Stories & Epics

## Epic 1 — Document ingestion

As a user, I want to connect my Google Drive and open PDFs so that my library persists across sessions.

## Epic 2 — Anchors & notes

As a user, I want to highlight text and create anchors so I can attach notes that never break.

## Epic 3 — Alignment

As a user, I want to align two documents so I can compare passages precisely.

## Epic 4 — AI audit

As a user, I want the AI to comment only on aligned passages, not hallucinate context.

## Epic 5 — Memory

As a user, I want the system to remember prior notes and alignments when I ask questions.
