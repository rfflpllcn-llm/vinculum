# Product Requirements Document — Vinculum

## Problem

Scholars working with parallel texts lack:

- persistent, shareable anchors  
- precise alignment across documents  
- AI assistance grounded in exact textual evidence

## Target users

- Literary scholars  
- Translators  
- Digital humanities researchers

## Core value

Vinculum provides *alignment-aware reading and reasoning* across documents.

## Core capabilities

- Google Drive–backed document library  
- PDF rendering with persistent anchors  
- Markdown notes linked to document coordinates  
- Dual-document alignment with sync scroll  
- AI audit and explanation tools  
- Long-term memory via vector search

## Out of scope

- Social annotation  
- Public publishing  
- OCR correction tooling
